# 📘 JavaScript Practice Tasks (Condition, Loop & Spread Operator)

## 🎯 Goal

Practice the following topics:

* Spread Operator (...)
* if, else, else if
* Nested if
* Shorthand if-else (Ternary Operator)
* for loop
* while loop
* do...while loop
* break
* continue

---

# 🧩 Task 1: Student Result Checker

Create a variable:

```js
let marks = 72;
```

Using **if, else if, else**, print the grade.

### Grade Rules

* 80 - 100 → A+
* 70 - 79 → A
* 60 - 69 → A-
* 50 - 59 → B
* 40 - 49 → C
* Below 40 → Fail

### Bonus

If the marks are less than 0 or greater than 100, print:

```
Invalid Marks
```

# 🧩 Task 2: Number Analyzer

Create a variable:

```js
let number = 45;
```

Using **Nested if**:

* Check whether the number is positive or negative.
* If it is positive:

  * Check whether it is even or odd.
* If it is negative:

  * Print "Negative Number".

### Example Output

```
Positive Even Number
```

or

```
Positive Odd Number
```

or

```
Negative Number
```

# 🧩 Task 3: Loop Practice

Using a **for loop**, print numbers from **1 to 30**.

Rules:

* Skip numbers divisible by 4 using **continue**.
* Stop the loop when the number becomes **25** using **break**.

### Expected Output

```
1
2
3
5
6
7
...
23
```

### Part B

# 🧩 Task 5: Spread Operator Challenge

Create two arrays:

```js
let frontend = ["HTML", "CSS", "JavaScript"];
let backend = ["Node.js", "Express", "MongoDB"];
```

Tasks:

* Merge the two arrays using the **Spread Operator**.
* Print the merged array.
* Add `"React"` at the beginning and `"Git"` at the end using the **Spread Operator**.
* Print the final array.

### Expected Output

```js
[
  "React",
  "HTML",
  "CSS",
  "JavaScript",
  "Node.js",
  "Express",
  "MongoDB",
  "Git"
]
```
